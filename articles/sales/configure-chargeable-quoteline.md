# <a name="configure-the-chargeable-components-of-a-project-based-quote-line"></a><span data-ttu-id="8f664-103">Configuració dels components imputables d'una línia d'oferta basada en projecte</span><span class="sxs-lookup"><span data-stu-id="8f664-103">Configure the chargeable components of a project-based quote line</span></span>

<span data-ttu-id="8f664-104">_**S'aplica a:** Project Operations per a escenaris basats en recursos/sense cotització_</span><span class="sxs-lookup"><span data-stu-id="8f664-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios_</span></span>

<span data-ttu-id="8f664-105">Una línia d'oferta basada en projecte pot incloure components i components imputables.</span><span class="sxs-lookup"><span data-stu-id="8f664-105">A project-based quote line can have included components and chargeable components.</span></span>

<span data-ttu-id="8f664-106">Els components inclosos estan subjectes al mètode de facturació, les divisions de client, els límits que no es poden superar i la configuració de la freqüència de factura definits a la línia d'oferta basada en projecte.</span><span class="sxs-lookup"><span data-stu-id="8f664-106">Included components are subject to the billing method, customer splits, not-to-exceed limits, and invoice frequency settings defined on the project-based quote line.</span></span>

   - <span data-ttu-id="8f664-109">Imputable</span><span class="sxs-lookup"><span data-stu-id="8f664-109">Chargeable</span></span>
   - <span data-ttu-id="8f664-110">No imputable</span><span class="sxs-lookup"><span data-stu-id="8f664-110">Non-chargeable</span></span>

<span data-ttu-id="8f664-111">Els components imputables es poden definir en funcions i categories de transacció.</span><span class="sxs-lookup"><span data-stu-id="8f664-111">Chargeable components can be defined on roles and transaction categories.</span></span>

<span data-ttu-id="8f664-112">La imputabilitat que es defineix en les funcions per a una línia d'oferta de projecte només s'aplica a la classe de transacció **Temps**.</span><span class="sxs-lookup"><span data-stu-id="8f664-112">Chargeability that is defined on roles for a project quote line only applies to the **Time** transaction class.</span></span> <span data-ttu-id="8f664-113">Si una línia d'oferta de projecte té **Inclou temps** = **No**, la pestanya **Funcions imputables** no està disponible.</span><span class="sxs-lookup"><span data-stu-id="8f664-113">If a project quote line has **Include Time** = **No**, the **Chargeable Roles** tab isn't available.</span></span>

<span data-ttu-id="8f664-114">La imputabilitat que es defineix en les categories de transacció d'una línia d'oferta de projecte només s'aplica a la classe de transacció **Despesa**.</span><span class="sxs-lookup"><span data-stu-id="8f664-114">Chargeability that is defined on transaction categories for a project quote line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="8f664-115">Si una línia d'oferta de projecte té **Inclou despeses** = **No**, la pestanya **Categories imputables** no està disponible.</span><span class="sxs-lookup"><span data-stu-id="8f664-115">If a project quote line has **Include Expenses** = **No**, the **Chargeable Categories** tab isn't available.</span></span>
