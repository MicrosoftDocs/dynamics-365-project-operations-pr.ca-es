# <a name="configuring-chargeable-components-of-a-project-based-contract-line"></a><span data-ttu-id="d9741-103">Configuració dels components imputables d'una línia de contracte basada en projectes</span><span class="sxs-lookup"><span data-stu-id="d9741-103">Configuring chargeable components of a project-based contract line</span></span>

<span data-ttu-id="d9741-104">_**S'aplica a:** implementació bàsica: tracte de facturació proforma_</span><span class="sxs-lookup"><span data-stu-id="d9741-104">_**Applies To:** Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="d9741-105">Una línia de contracte basada en projectes té components *inclosos* i components *imputables*.</span><span class="sxs-lookup"><span data-stu-id="d9741-105">A project-based contract line has *included* components and *chargeable* components.</span></span>

<span data-ttu-id="d9741-106">Els components inclosos són els que estan subjectes a:</span><span class="sxs-lookup"><span data-stu-id="d9741-106">Included components are components that are subject to:</span></span>

  - <span data-ttu-id="d9741-107">Mètode de facturació i desglossament del client</span><span class="sxs-lookup"><span data-stu-id="d9741-107">Billing method and customer splits</span></span>
  - <span data-ttu-id="d9741-108">Límits que no s'han de superar</span><span class="sxs-lookup"><span data-stu-id="d9741-108">Not-to-exceed limits</span></span> 
  - <span data-ttu-id="d9741-109">Paràmetres de freqüència de facturació definits a la línia de contracte basada en projectes</span><span class="sxs-lookup"><span data-stu-id="d9741-109">Invoice frequency settings defined on the project-based contract line</span></span>

<span data-ttu-id="d9741-110">Un subconjunt dels components inclosos es pot marcar com a imputable mitjançant el camp **Tipus de facturació**.</span><span class="sxs-lookup"><span data-stu-id="d9741-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="d9741-111">El camp **Tipus de facturació** és un conjunt d'opcions que permet els següents valors en el context d'una línia de contracte:</span><span class="sxs-lookup"><span data-stu-id="d9741-111">The **Billing Type** field is an option-set that allows the following values in the context of a contract line:</span></span>

  - <span data-ttu-id="d9741-112">Imputable</span><span class="sxs-lookup"><span data-stu-id="d9741-112">Chargeable</span></span>
  - <span data-ttu-id="d9741-113">No imputable</span><span class="sxs-lookup"><span data-stu-id="d9741-113">Non-chargeable</span></span>

<span data-ttu-id="d9741-114">Els components imputables es poden definir en tasques, rols i categories de transacció.</span><span class="sxs-lookup"><span data-stu-id="d9741-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="d9741-115">La imputabilitat es defineix en les tasques d'una línia de contracte de projecte i s'aplica a totes les classes de transacció incloses a la línia.</span><span class="sxs-lookup"><span data-stu-id="d9741-115">Chargeability is defined on tasks for a project contract line and applies to all transaction classes included on the line.</span></span> <span data-ttu-id="d9741-116">Si el camp **Inclou les tasques** d'una línia de contracte és buit o s'ha establert a **Tot el projecte** , la pestanya **Tasques imputables** no estarà disponible.</span><span class="sxs-lookup"><span data-stu-id="d9741-116">If the **Include Tasks** field on a contract line is blank or set to \* *Entire project* , the **Chargeable tasks** tab will not be available.</span></span>

<span data-ttu-id="d9741-117">La imputabilitat definida a les funcions d'una línia de contracte de projecte només s'aplica a la classe de transacció **Temps**.</span><span class="sxs-lookup"><span data-stu-id="d9741-117">Chargeability defined on roles for a project contract line only applies to the **Time** transaction class.</span></span> <span data-ttu-id="d9741-118">Si el camp **Inclou el temps** d'una línia de contracte és buit o s'ha establert a **No** , la pestanya **Funcions imputables** no estarà disponible.</span><span class="sxs-lookup"><span data-stu-id="d9741-118">If the **Include Time** field on a contract line is set to **No** , the **Chargeable roles** tab will not be available.</span></span>

<span data-ttu-id="d9741-119">La imputabilitat definida a les categories de transacció d'una línia de contracte de projecte només s'aplica a la classe de transacció **Despesa**.</span><span class="sxs-lookup"><span data-stu-id="d9741-119">Chargeability defined on transaction categories for a project contract line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="d9741-120">Si el camp **Inclou les despeses** s'ha establert a **No** , la pestanya **Categories imputables** no estarà disponible.</span><span class="sxs-lookup"><span data-stu-id="d9741-120">If the **Include Expenses** field is set to **No** , the **Chargeable Categories** tab will not be available.</span></span>

### <a name="update-a-project-task-as-chargeable-or-non-chargeable"></a><span data-ttu-id="d9741-121">Actualitzar una tasca de projecte com a imputable o no imputable</span><span class="sxs-lookup"><span data-stu-id="d9741-121">Update a project task as chargeable or non-chargeable</span></span>

<span data-ttu-id="d9741-122">Una tasca de projecte pot ser imputable o no imputable en una línia de contracte específica que fa possible la següent configuració:</span><span class="sxs-lookup"><span data-stu-id="d9741-122">A project task can be chargeable or non-chargeable on a specific contract line which makes the following setup possible:</span></span>

<span data-ttu-id="d9741-123">Si una línia de contracte basada en el projecte inclou **Temps** i una tasca determinada, **T1** s'hi associa com a imputable.</span><span class="sxs-lookup"><span data-stu-id="d9741-123">If a project-based contract line includes **Time** and a certain task, **T1** is associated to it as chargeable.</span></span> <span data-ttu-id="d9741-124">Si hi ha una segona línia de contracte que inclou **Despesa** , es pot associar la tasca T1 a la línia de contracte com a no imputable.</span><span class="sxs-lookup"><span data-stu-id="d9741-124">If there is a second contract line that includes **Expense** , you can associate the T1 task on the contract line as non-chargeable.</span></span> <span data-ttu-id="d9741-125">El resultat és que tot el temps que es registra en la tasca és imputable i totes les despeses no són imputables.</span><span class="sxs-lookup"><span data-stu-id="d9741-125">The result is that all of the time recorded on the task is chargeable and all expenses are non-chargeable.</span></span>

<span data-ttu-id="d9741-126">El tipus de facturació d'una tasca es pot configurar a la pestanya **Tasques imputables** de la línia de contracte actualitzant el camp **Tipus de facturació** a la subquadrícula de les tasques de la línia de contracte.</span><span class="sxs-lookup"><span data-stu-id="d9741-126">A task's billing type can be configured on the **Chargeable Tasks** tab of the contract line by updating the **Billing Type** field on the contract line tasks sub-grid.</span></span> <span data-ttu-id="d9741-127">Alternativament, podeu actualitzar el camp **Tipus de facturació** a la subquadrícula de la tasca Configuració de facturació d'un projecte que mostra les línies de contracte associades a una tasca.</span><span class="sxs-lookup"><span data-stu-id="d9741-127">Alternatively, you can update the **Billing Type** field on the sub-grid of the task Billing setup of a project that shows the contract lines associated to a task.</span></span>

<span data-ttu-id="d9741-129">Una funció pot ser imputable o no imputable en una línia de contracte específica.</span><span class="sxs-lookup"><span data-stu-id="d9741-129">A role can be chargeable or non-chargeable on a specific contract line.</span></span>

<span data-ttu-id="d9741-134">El tipus de facturació d'una transacció es pot configurar a la pestanya **Categories imputables** d'una línia de contracte basada en projectes.</span><span class="sxs-lookup"><span data-stu-id="d9741-134">A transaction's billing type can be configured on the **Chargeable Categories** tab of a project-based contract line.</span></span> <span data-ttu-id="d9741-135">Per fer-ho, actualitzeu el camp **Tipus de facturació** a la subquadrícula **Categories imputables**.</span><span class="sxs-lookup"><span data-stu-id="d9741-135">To do this, update the **Billing Type** field on the **Chargeable Categories** sub-grid.</span></span>
